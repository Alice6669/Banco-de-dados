# Banco de Dados - Exercícios

Repositório dedicado a exercícios e estudos de **Banco de Dados** com foco em diferentes plataformas e tecnologias
de gerenciamento de dados.

## 📚 Conteúdo

Este repositório contém exercícios práticos e exemplos de código para as seguintes tecnologias:

### 🐬 MySQL
Exercícios e scripts SQL utilizando **MySQL**, incluindo:
- Criação e manipulação de tabelas
- Consultas SELECT, INSERT, UPDATE, DELETE
- Relacionamentos e constraints
- Índices e otimização

### 🖥️ SQL Server
Exercícios e scripts T-SQL utilizando **SQL Server**, incluindo:
- Consultas avançadas com T-SQL
- Stored Procedures
- Triggers e views
- Performance tuning

## 📊 Mapa Conceitual

Este repositório inclui um **mapa conceitual** que organiza visualmente os conceitos e relacionamentos de banco de dados.

### ⚠️ Como Abrir o Mapa Conceitual

Para visualizar o mapa conceitual corretamente, você **DEVE** utilizar o aplicativo **BR Modelo**:

1. Baixe o **BR Modelo** em: [https://www.brmodeloweb.com/](https://www.brmodeloweb.com/)
2. Abra o aplicativo BR Modelo
3. Importe o arquivo do mapa conceitual através do menu `Arquivo > Abrir`
4. O mapa será exibido com todos os relacionamentos e entidades

> **Nota:** O BR Modelo é a ferramenta padrão para visualizar e editar modelos conceituais de banco de dados em
> português, garantindo compatibilidade completa com os arquivos deste repositório.
